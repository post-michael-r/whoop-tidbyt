# Privacy Policy

This is a personal-use application built and operated by a single user (the developer). It is not offered as a service to anyone else.

## What it does

- Reads the user's own Whoop strain data via the official [Whoop API](https://developer.whoop.com/).
- Renders that data and displays it on the user's own [Tidbyt](https://tidbyt.com/) device.

## What it does not do

- It does not store user data in a database or persistent backend.
- It does not share or transmit data to any third party other than the user's own Tidbyt device for the purpose of displaying it.
- It does not include analytics, tracking, or telemetry.

## Credentials

API credentials (Whoop and Tidbyt) are stored locally in environment variables on the user's own machine. They are used only to authenticate with the respective official APIs and are not transmitted anywhere else.
